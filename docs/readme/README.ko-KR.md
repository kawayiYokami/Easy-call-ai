Easy Call AI는 일상 PC 작업 흐름에 맞춘 데스크톱 AI 어시스턴트입니다.
단순한 채팅 웹앱이 아니라, 단축키 한 번으로 불러와 바로 문제를 해결하는 사용성을 목표로 합니다.

## 왜 만들었나

> 많은 AI 도구가 강력하지만, 실제 사용에서는 다음과 같은 불편이 발생합니다:

| 불편 사항 | 설명 |
|-----------|------|
| 잦은 전환 | 브라우저 및 앱 간 전환이 잦아 작업 흐름이 끊김 |
| 설정 분산 | 공급자 및 모델 설정이 흩어져 관리가 어려움 |
| 대화 상실 | 긴 대화가 지나면 맥락과 기록 추적이 어려움 |
| 경험 분리 | 데스크톱 환경과 AI 사용 경험이 분리되어 있음 |

**Easy Call AI**는 이러한 마찰을 줄여 AI를 데스크톱의 기본 기능처럼 사용할 수 있게 합니다.

## 대상 사용자

- 작업 중 AI에게 자주 질문하는 사용자
- 여러 공급자 및 모델을 함께 사용하는 사용자
- 필요할 때만 빠르게 호출해서 사용하고 싶은 사용자
- 대화 기록과 기억을 장기적으로 활용하고 싶은 사용자

## 주요 기능

- **전역 단축키** — 대화창 표시/숨김
- **트레이 상주** — 설정 / 대화 / 아카이브 / 종료
- **다중 LLM 설정** — 여러 공급자 및 모델 설정 저장
- **AI 모드 분리** — 대화 AI와 이미지 해석 AI 독립 설정
- **도구 호출** — 검색 / 가져오기 / 기억
- **다중 페르소나** — 다양한 AI 성격 전환
- **스트리밍 응답** — 실시간 응답 및 사고 내용 표시
- **이미지 붙여넣기** — 멀티모달 메시지 저장
- **자동 아카이브** — 추적 가능한 대화 기록
- **다국어 UI** — 중 / 영 / 일 / 한

## 대표 사용 시나리오

- 문서, 오류, 스크린샷을 볼 때 바로 질문
- 작업 성격에 따라 모델을 즉시 전환
- 긴 대화를 자동 아카이브로 정리
- 기억 기능으로 연속 대화 품질 향상

## 빠른 시작

<details>
<summary>사용 방법 펼치기</summary>

> 실행 후 화면 오른쪽 하단 시스템 트레이를 확인하세요. 앱 아이콘이 상주하며, 우클릭으로 설정 창을 열 수 있습니다.

1. 앱 실행 후 설정 창을 엽니다
2. `LLM` 탭에서 API 설정을 추가하고 저장합니다
3. `대화` 탭에서 대화AI / 이미지 해석AI / AI 페르소나를 선택합니다
4. 호출 단축키로 대화창을 열고 질문을 시작합니다
5. 지난 대화는 아카이브 창에서 확인합니다

</details>

## 개인정보 및 데이터

- API Key와 설정은 로컬 장치에 저장됩니다
- 대화, 아카이브, 기억 데이터도 기본적으로 로컬 저장입니다
- 캐시 정리, 아카이브 삭제, 데이터 납품을 지원합니다
