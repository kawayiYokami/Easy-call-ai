# Easy Call AI

[中文](../../README.md) | [English](README.en-US.md) | [日本語](README.ja-JP.md) | [한국어](README.ko-KR.md)

Easy Call AI는 일상적인 PC 작업 흐름에 맞춘 데스크톱 AI 어시스턴트입니다.  
단순한 채팅 웹앱이 아니라, 단축키 한 번으로 불러와 바로 문제를 해결하는 사용성을 목표로 합니다.

## 왜 만들었나

많은 AI 도구가 강력하지만, 실제 사용에서는 이런 불편이 자주 발생합니다.

- 브라우저/앱 전환이 잦아 작업 흐름이 끊김
- 공급자/모델 설정이 흩어져 관리가 어려움
- 대화가 길어지면 맥락과 기록을 추적하기 어려움
- 데스크톱 환경과 AI 사용 경험이 분리되어 있음

Easy Call AI는 이 마찰을 줄여 AI를 데스크톱의 기본 기능처럼 쓰게 하려는 프로젝트입니다.

## 이런 사용자에게 적합

- 작업 중 AI에게 자주 질문하는 사용자
- 여러 공급자/모델을 함께 쓰는 사용자
- 필요할 때만 빠르게 호출해서 쓰고 싶은 사용자
- 대화 기록과 기억을 장기적으로 활용하고 싶은 사용자

## 주요 기능

- 전역 단축키로 대화창 표시/숨김
- 트레이 상주(설정 / 대화 / 아카이브 / 종료)
- 다중 LLM 설정 관리
- 대화 AI와 이미지 해석 AI 분리 설정
- 도구 호출(검색 / 가져오기 / 기억)
- 다중 페르소나 전환
- 스트리밍 응답 및 사고 내용 표시
- 이미지 붙여넣기와 멀티모달 메시지 저장
- 대화 자동 아카이브 및 아카이브 뷰어
- 기억 데이터 가져오기/내보내기
- UI 다국어(중 / 영 / 일 / 한)

## 대표 사용 시나리오

- 문서/에러/스크린샷을 보면서 바로 질문
- 작업 성격에 따라 모델을 즉시 전환
- 긴 대화를 자동 아카이브로 정리
- 기억 기능으로 연속 대화 품질 향상

## 사용 방법(사용자 기준)

1. 앱 실행 후 설정 창을 엽니다
2. `LLM` 탭에서 API 설정을 추가하고 저장합니다
3. `대화` 탭에서 대화AI / 이미지 해석AI / AI 페르소나를 선택합니다
4. 호출 단축키로 대화창을 열고 질문을 시작합니다
5. 지난 대화는 아카이브 창에서 확인합니다

## 개인정보 및 데이터

- API Key와 설정은 로컬 장치에 저장됩니다
- 대화/아카이브/기억 데이터도 기본적으로 로컬 저장입니다
- 캐시 정리, 아카이브 삭제, 데이터 내보내기를 지원합니다
