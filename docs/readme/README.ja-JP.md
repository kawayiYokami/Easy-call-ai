# Easy Call AI

[中文](../../README.md) | [English](README.en-US.md) | [日本語](README.ja-JP.md) | [한국어](README.ko-KR.md)

Easy Call AI は、日常の PC 作業に溶け込むデスクトップ AI アシスタントです。  
「チャットサイトをもう 1 つ作る」のではなく、ホットキー 1 つで呼び出して、すぐに問題を解決することを目的にしています。

## 開発の背景

多くの AI ツールは高機能ですが、実運用では次のような課題があります。

- ブラウザやアプリ間の切り替えが多く、作業が途切れる
- プロバイダやモデル設定が分散し、管理しにくい
- 会話が長くなると履歴が追いづらい
- デスクトップ作業と AI 体験が分断されている

Easy Call AI は、こうした摩擦を減らし、AI をデスクトップの標準機能のように使える体験を目指しています。

## こんな人に向いています

- 作業中に AI へ頻繁に質問する人
- 複数プロバイダ / 複数モデルを使い分ける人
- 必要なときだけ素早く呼び出して使いたい人
- 会話履歴や記憶を継続して活用したい人

## 主な機能

- グローバルホットキーで会話ウィンドウを表示 / 非表示
- トレイ常駐（設定 / 会話 / アーカイブ / 終了）
- 複数 LLM 設定の管理
- 会話 AI と画像読解 AI の分離設定
- ツール呼び出し（検索 / 取得 / 記憶）
- 複数人格の切り替え
- ストリーミング応答と思考表示
- 画像貼り付けとマルチモーダル保存
- 会話の自動アーカイブと履歴閲覧
- 記憶データのインポート / エクスポート
- UI 多言語対応（中 / 英 / 日 / 韓）

## 典型的な使い方

- ドキュメント・エラー・スクリーンショットをその場で質問
- タスクに応じてモデルを素早く切り替え
- 長い会話を自動アーカイブして整理
- 記憶機能で継続的なやり取りの品質を向上

## 使い始め（ユーザー向け）

1. アプリ起動後、まず設定ウィンドウを開く
2. `LLM` タブで API 設定を追加して保存
3. `会話` タブで「会話AI / 画像読解AI / AI人格」を選択
4. 呼び出しホットキーで会話ウィンドウを開いて利用開始
5. 過去の内容はアーカイブウィンドウで確認

## プライバシーとデータ

- API Key と設定はローカル端末に保存されます
- 会話 / アーカイブ / 記憶データも既定でローカル保存です
- キャッシュ削除、アーカイブ削除、データエクスポートが可能です
